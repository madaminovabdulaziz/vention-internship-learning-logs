## Structure

Each day lives in its own `day-XX-topic/` folder. Inside, topics are broken into numbered subfolders (`01-...`, `02-...`), each with the same three-file split:

- `playground.py` — scratch code, experiments, things I tried while learning
- `exercises.py` — practice problems and my solutions
- `notes.md` — written notes, takeaways, and gotchas
